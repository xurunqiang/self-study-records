# self-study-records
大二那会儿就铁了心要啃透Transformer这块硬骨头，结果愣是被知识储备不足、畏难摆烂、没耐心这仨“拦路虎”按在地上摩擦，最后只能灰溜溜放弃，心里总搁着个疙瘩。
熬到研一来到电科，被这儿卷到飞起的学习氛围一激，那股不服输的斗志又燃起来了！可现实依旧骨感，日常通勤的奔波+专业课的压力，愣是把学习Transformer的时间挤得七零八落，折腾半天还是一头雾水。
好不容易盼来寒假，这次总算找对路子——把Transformer拆解开塞到Jupyter里逐行啃，还把能扒拉出来的参数、矩阵全扒拉成热力图可视化，主打一个“把抽象变直观”。第一遍啃还是满脑子问号，索性拉着完整代码从头debug，对着Jupyter的内容一一验证。咱主打一个“脑子不够，可视化来凑”，把各路参数扒得明明白白，结合AI刨根问底：这参数干啥的？作用是啥？作者当初这么设计到底咋想的？
死磕下来，居然真把Transformer的架构摸透了，各个细节也都门儿清！缠了我好几年的夙愿，总算圆满了结，那叫一个爽！
也真心推荐学Transformer的新手试试这法子：别硬啃公式，把每个参数的来龙去脉揪出来掰扯清楚，结合AI搞懂设计逻辑，多琢磨作者的巧思，把基础打牢了，再谈超越和创新。其实Transformer也不是啥三头六臂的大神，找对方法死磕到底，咱普通人也能啃下来～
